# HRIS EVERGREEN

A Human Resources Information System built with PHP and MySQL.

## Features

- **Employee Management** – Full CRUD operations for employee records
- **Attendance Tracking** – Time-in/out with automatic hour calculations
- **Leave Management** – Request, approve, and track employee leaves
- **Recruitment** – Job posting and applicant tracking
- **Role-Based Access** – Admin, HR Manager, Supervisor, and Employee roles

## Quick Start

1. **Setup XAMPP** – Place project in `C:\xampp\htdocs\hris-sia`
2. **Database** – Import SQL schema via phpMyAdmin into `BankingDB (if not working, pls check "Evergreen" repo db)`
3. **Configure** – Update credentials in `config/database.php`
4. **Access** – Navigate to `http://localhost/hris-sia`

## Stack

- PHP 8+ / MySQL / Tailwind CSS / Chart.js
